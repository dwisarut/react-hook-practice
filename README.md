# React Hook Practice

Well, this repo is my exercise by making some components and practice the react hooks, with reviewing vanilla CSS just for the sake of it.

## Getting started

1. Clone this repository

    ```
    git clone https://github.com/dwisarut/finocular.git
    cd finocular
    ```
2. Install dependencies

    ```
    npm install
    ```
3. Run this project

    ```
    npm run dev
    ```
